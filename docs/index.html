Descripción

Go Bug It! es el nuevo éxito de Ñintendo aclamado por la crítica como "El mejor juego de la generación actual de videojuegos".

Go Bug It! es un puzzle-plataformer en el que necesitarás usar tanto tu cabeza como tus reflejos para poder avanzar entre una variedad de antivirus de lo más molestos, ya que en este juego ¡TU ERES EL VIRUS!
